# Product Context — cafe_racer_93_3d

## Qué es
**Cafe Racer 93** es un juego web tipo *highway racer* con estética **synthwave**, en **3D real (WebGL / three.js)**. El jugador pilotea una moto café racer de neón por una autopista nocturna infinita, esquiva tráfico y acumula puntaje por distancia, velocidad y casi-choques.

## Requerimiento original
> Juego tipo highway racer con estética synthwave, interfaz profesional, listo para producción.
> Esta versión lo lleva a **3D**: la moto del jugador es un **modelo 3D generado con Higgsfield** y la escena synthwave es una escena 3D (no un canvas 2D).

## Para quién / cómo se usa
- Jugable en navegador (desktop + móvil + gamepad). Sesiones de minutos, score-attack (batir el récord).
- Ya está **desplegado y jugable**: https://tender-beam-198.higgsfield.gg/
- Engagement principal: **ejecución** (reflejos para esquivar) + **acumulación** (récord personal).
